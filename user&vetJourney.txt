User Journey
1. เข้าหน้า login
2. สร้างบัญชี
    2.1 ถ้าใส่ชื่อไม่ครบ หรือรหัสผ่านไม่ตรงกัน จะมีขึ้นแจ้งเตือน
    2.2 ถ้าใส่อีเมลที่เคยสมัครแล้วจะขึ้นแจ้งเตือน
    2.3 เมื่อกดสร้างบัญชีจะกลับมาที่หน้า login
3. ทำการ login
    3.1 ถ้าใส่อีเมลหรือรหัสไม่ถูก จะมีขึ้นแจ้งเตือน
4. เมื่อ login ก็จะเข้ามายังหน้า HomeScreen โดยจะแสดงอีเมลล์ รูปโปรไฟล์ และจำนวนเงินภายในแอพของเราที่บาร์ด้านบน
5. ในหน้า HomeScreen จะสามารถกดไปได้อีก 2 หน้าคือ หน้า  SelectVetScreen และ ProfileScreen
6. เมื่อเข้ามาหน้า SelectVetScreen จะแสดงรายชื่อหมอของ sopet
    6.1 เราสามารถกดพิม search และเลือกปุ่ม filter ได้
7. เมื่อเข้ามาหน้า ProfileScreen จะแสดงชื่อ รูปโปรไฟล์ และ จำนวนเงินของเรา
    7.3 สามารถกดไปหน้าสัตว์เลี้ยงของ userได้
        7.3.1 เมื่อกดปุ่ม "บวก" จะสามารถใส่ข้อมูลเพื่อเพิ่มสัตว์เลี้ยงใหม่ได้
         7.41 สามารถแก้ไขข้อมูลสัตว์เลี้ยงได้ ซึ่งยังไม่ทำการเชื่อม front กับ back
8. สามารถกดออกจากระบบได้

Vet Journey
1. เข้าหน้า login
2. เข้าหน้า login ของ vet
3. สร้างบัญชีของ  Vet
    3.1 ถ้าใส่ชื่อไม่ครบ หรือรหัสผ่านไม่ตรงกัน จะมีขึ้นแจ้งเตือน
    3.2 ถาใสอีเมลทเคยสมัครแล้วจะขนแจ้งเตือน
    3.3 เมื่อกดสรางบญชจะกลบมาทหนา login
4. เมอเขามากจะเขามาทหนา VetHomeScreen
5. ในหนา VetHomeScreen จะสามารถกดไปไดอกหนาคอ VetProfileScreen
6.เมอเขามาหนา VetProfileScreen จะเหนขอมล และ รวว ของ Vet
    6.1 สามารถกด edit เพอแกไขโปรไฟลของ Vet ได
